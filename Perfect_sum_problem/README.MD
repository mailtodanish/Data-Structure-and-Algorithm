- JavaScript_perfect_sum_problem

All subsets of the given array with the sum equal to the given number.

## INPUT:

```
[5,1, 2, 3,4]
```

Possible Combination : 5P5 + 5P4 + 5P3 + 5P2 + 5P1

## OUTPUT:

```
5+1+2=8
5+2+1=8
5+3=8
1+5+2=8
1+2+5=8
1+3+4=8
1+4+3=8
2+5+1=8
2+1+5=8
3+5=8
3+1+4=8
3+4+1=8
4+1+3=8
4+3+1=8
```
